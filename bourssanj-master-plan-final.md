# پلن نهایی بورس‌سنج — ترکیب نظر جمنای، چت‌جی‌پی‌تی، کلاد

> منبع: سه تحلیل مستقل (Gemini، ChatGPT، Claude) روی رقبا (فاندبیس، تریدرزآرما، بورس‌ترید، ره‌آورد۳۶۵، بورس‌ویو، TSETMC، TGJU) + وضعیت فعلی کدبیس.
> اصل حاکم: جمنای می‌گفت فیچرهای رقبا رو سریع بساز؛ چت‌جی‌پی‌تی می‌گفت کل جایگاه رو به «AI تصمیم‌یار» تغییر بده؛ کلاد می‌گفت اول روی حباب صندوق کالایی + خلاصه‌ساز کدال عمیق شو. هر سه مستقل به یک نتیجه رسیدن: **خلاصه‌سازی هوشمند کدال، فیچریه که هیچ رقیبی نداره و باید اولویت اول باشه.**

⚠️ **قبل شروع هر فاز**: خیلی از آیتم‌های «فاز ۱» رقبا در بورس‌سنج از قبل ساخته شده (بعضی دیپلوی نشده). چک کن قبل کار تکراری:
- هشدار قیمت/حباب + cron سرور → ساخته شده (۲۰۲۶-۰۷-۱۷)
- رتبه‌بندی سیگنال + بک‌تست (`signal_backtest_stats`) → ساخته شده، تازه SMC (BOS/CHoCH/OB/FVG) هم اضافه شد
- مقایسه چندنمادی سهام، Export CSV → ساخته شده
- ربات تلگرام پرتفوی (@bsportfo_bot) → زنده
- Codal auto-update (هر ۳۰ دقیقه، بدون rebuild) → زنده
- Portfolio /optimize (min-variance) → ساخته شده، **دیپلوی نشده**
- Portfolio live monitor (stock_watch_5m) → ساخته شده، **دیپلوی نشده**
- Candles + ۵ روش تعدیل قیمت → ساخته شده، **دیپلوی نشده**
- Anomaly-watch + گزارش تلگرام (سهام+طلا/نقره) → زنده

## فاز ۰: تثبیت (۱ تا ۲ هفته)

قبل فیچر جدید، کارهای نیمه‌کاره رو ببند — وگرنه بدهی فنی روی بدهی فنی جمع می‌شه:
1. دیپلوی Portfolio live monitor, Portfolio /optimize, Candles pipeline (۵ روش تعدیل) روی سرور.
2. نتیجه ۱۰روزه track-record سیگنال سهام رو جمع کن، منتشرش کن به‌عنوان کارت شفافیت عمومی (این دقیقاً همون چیزیه که هر سه گزارش گفتن «بزرگ‌ترین اعتمادساز»).
3. PostHog env vars روی سرور ایران (.env سرور 45.94.215.115) ست کن (آنالیتیکس واقعی لازمه که بفهمی فاز بعد چی جواب داد).

## فاز ۱: برد سریع (۱ تا ۲ ماه)

هدف: چیزی که رقبا دارن و بورس‌سنج نداره، ولی هزینه ساختش کمه.

1. **خلاصه‌ساز AI گزارش کدال — نسخه ارتقایافته** (نقطه مشترک هر سه گزارش، اولویت مطلق):
   - فرمت ثابت: مثبت/منفی + تأثیر تخمینی EPS + یک جمله «یعنی چی برای سهام‌دار».
   - چون LLM narration از قبل هست، این کار توسعه نه ساخت از صفر — فرمت خروجی رو استاندارد کن.
2. **صفحه اختصاصی هر صندوق کالایی** (طلا/نقره/زعفران): NAV لحظه‌ای + حباب اسمی/ذاتی/معاملاتی + تفکیک سکه/شمش + ترکیب دارایی + نمودار تاریخچه حباب ۳۰/۹۰/۳۶۵روزه. این دقیقاً wedge کلاد — جایی که هیچ رقیبی جدی نیست.
3. **پارامترهای تابلوخوانی روی دیده‌بان‌های موجود**: سرانه خرید/فروش حقیقی، قدرت خریدار به فروشنده. (نه ریزمعاملات کامل — اون هزینه‌بره، فاز ۲.)
4. **صفحه‌های فرود سئو** برای عبارات پرجست‌جو («حباب صندوق طلا امروز»، «حباب سکه»، «NAV صندوق X») — کلمات کلیدی الان دست فاندبیس/TGJU‌ست.

---

## فاز ۲: تمایز عمیق (۳ تا ۴ ماه)

هدف: چیزی بساز که رقبا اصلاً تلاش هم نکردن یا بد انجام دادن.

1. **باکس نظرات/کامنت زیر هر نماد و صندوق** (نکته جمنای — ره‌آورد ترافیکش رو مدیون همینه). شروع محدود: کامنت ساده + مدیریت اسپم، نه انجمن کامل.
2. **بک‌تست تعاملی سیگنال** (کاربر روی نمادهای دلخواه بسنجه) — روی زیرساخت `signal_backtest_stats` موجود بساز.
3. **شاخص هم‌وزن صندوق‌های طلا/نقره بورس‌سنج** — نسخه اختصاصی، رقبا هم‌چین چیزی ندارن.
4. **دستیار زبان طبیعی برای فیلترکردن** («صندوق‌های طلا با حباب منفی و نقدشوندگی بالا») — روی `/api/chat` و `assistant_qa_log` موجود گسترش بده، نه سیستم جدید.
5. **رادار پول هوشمند صندوق‌ها** — تشخیص ورود/خروج غیرعادی پول به صندوق‌های کالایی (نمونه‌ی anomaly-watch موجود، ولی مختص صندوق).

---

## فاز ۳: لایه AI/Copilot (۵ تا ۸ ماه)

اینجا چشم‌انداز بزرگ چت‌جی‌پی‌تی وارد می‌شه — ولی به‌ترتیب، نه یک‌جا:

1. **Regime Engine ساده**: هر روز بازار یک برچسب بگیره (صعودی/نزولی/نوسانی/تجمیع) بر اساس داده‌های موجود (شاخص، ورود پول، حجم). شروع rule-based، نه ML پیچیده.
2. **Market Story خودکار روزانه**: «چرا امروز بازار/طلا/سکه این‌طور شد» — خروجی متنی از LLM narration موجود + Regime Engine، منتشر در سئو + تلگرام.
3. **Portfolio Intelligence**: تحلیل ریسک/تمرکز پرتفوی کاربر (روی زیرساخت /optimize و live monitor موجود بساز) + هشدار «وزن صنعت فلان دو برابر میانگین بازاره».
4. **Copilot محدود** («بین صندوق X و Y کدوم بهتره؟») — نسخه اول محدود به صندوق‌های کالایی (دیتای عمیق‌ترین بخش شما)، نه کل بازار.

⚠️ **Market Memory** (چت‌جی‌پی‌تی): ایده خوبیه ولی نیاز به دیتای تاریخی صحت پیش‌بینی داره — با شفافیت بک‌تست فاز ۱/۲ به‌طور طبیعی زیرساختش جمع می‌شه، جلوتر از فاز ۳ سراغش نرو.

## فاز ۴: مقیاس و درآمد (بلندمدت، بعد از اعتبارسنجی فاز ۱-۲)

1. Freemium: دیتای پایه رایگان، هشدار آنی + بک‌تست کامل + پرتفوی هوشمند → VIP.
2. Export API برای تحلیلگر/سبدگردان حرفه‌ای.
3. PWA / اپ موبایل (فقط بعد اثبات تقاضا — قبلاً یک‌بار رد شده در بک‌لاگ رشد).
4. ریزمعاملات کامل (Tick Data) و بوم TradingView — هزینه/پیچیدگی بالا، فقط اگه کاربر حرفه‌ای/day trader واقعاً هدف بازاره.

---

## سه حرکت تعیین‌کننده (اگه فقط ۳ تا انتخاب کنی)

1. **خلاصه‌ساز کدال با فرمت استاندارد (مثبت/منفی + EPS + یعنی‌چی)** — قابل کپی نیست، رقیب جدی نداره.
2. **ترک‌رکورد عمومی سیگنال‌ها** — بزرگ‌ترین اعتمادساز، تبلیغ رایگان.
3. **صفحه اختصاصی صندوق کالایی + کامنت** — سئو رایگان از فاندبیس/TGJU می‌گیره، و درگیری کاربر می‌سازه (نکته جمنای).

## KPI پیگیری

- نرخ بازگشت روزانه (DAU/retention) بعد فعال‌سازی هشدار.

- نرخ استفاده از بک‌تست تعاملی و ترک‌رکورد (اعتماد).
- تعداد کامنت/تعامل زیر صفحات صندوق.
- نرخ تبدیل رایگان → VIP (فقط بعد فاز ۴).
